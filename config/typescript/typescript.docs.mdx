---
description: "A basic typescript config"
labels: ["typescript", "config", "morajlab", "moraj", "tsconfig", "ts-node"]
---

Usage:

```json
{
  "extends": "/path/to/node_modules/@morajlab/config.typescript/tsconfig.base.json",
  "ts-node": {
    "compilerOptions": {
      "module": "commonjs"
    }
  }
}
```

for `ts-node` config:

```json
{
  "extends": "/path/to/node_modules/@morajlab/config.typescript/ts-node.json"
}
```
